# <a name="help-signing-in-with-windows-hello-pin"></a>Ajuda para entrar com o PIN do Windows Hello

- Para mudar, redefinir ou recuperar o PIN (quando conectado), visite [Alterar seu PIN quando você já estiver conectado ao seu dispositivo](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19).

- Para mudar, redefinir ou recuperar o PIN (quando sair), visite [Redefinir seu PIN quando não estiver conectado no Windows 10](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904).

- Para configurar o PIN do Windows Hello, visite [Saiba mais sobre o Windows Hello e como configurá-lo](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0).

# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Solução de Outlook configuração móvel para uma caixa de correio compartilhada

Não é possível acessar a caixa de correio compartilhada do aplicativo Outlook que você instalou no dispositivo móvel. Quer esse recurso? Avise a equipe Outlook Engenharia. [Vote aqui](https://go.microsoft.com/fwlink/?linked=862116)!
  
Você só pode usar uma caixa de correio compartilhada em um dispositivo móvel abrindo seu navegador, conectado e indo para Outlook na Web. Para obter instruções, [consulte Add a shared mailbox in Outlook na Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Em Outlook na Web, você poderá acessar a caixa de correio compartilhada.
  
Saiba mais: [Obter ajuda no aplicativo para Outlook iOS e Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)

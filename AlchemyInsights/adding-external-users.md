# <a name="adding-external-users"></a>Adicionar usuários externos

As etapas para adicionar usuários externos/convidados podem ser diferentes, dependendo do que você precisa que eles possam acessar. Escolha um dos seguintes:
  
- [SharePoint Visão geral do compartilhamento externo online](https://docs.microsoft.com/sharepoint/external-sharing-overview)

- [Acesso de convidados em Microsoft 365 Grupos](https://support.office.com/article/guest-access-in-office-365-groups-bfc7a840-868f-4fd6-a390-f347bf51aff6)

- [Acesso para convidado no Microsoft Teams](https://docs.microsoft.com/microsoftteams/guest-access-checklist)

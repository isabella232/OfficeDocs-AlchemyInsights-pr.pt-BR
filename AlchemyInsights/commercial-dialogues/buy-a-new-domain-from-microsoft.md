# <a name="buy-a-new-domain-from-microsoft"></a>Comprar um novo domínio da Microsoft

Para comprar um novo domínio da Microsoft, execute as seguintes etapas:

1. Navegue **até Configurar > Domínios e** clique em Comprar **domínio**. 
2. Siga as instruções do assistente.

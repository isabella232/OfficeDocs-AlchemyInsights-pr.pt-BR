# <a name="block-downloads-for-view-only-files-in-sharepoint-and-onedrive"></a>Bloquear downloads de arquivos somente exibição no SharePoint e no OneDrive

Ao compartilhar itens com outras pessoas no OneDrive ou no SharePoint, você pode conceder a eles acesso para exibição, mas não para download. Para saber mais, confira [Bloquear downloads de arquivos somente exibição no SharePoint e no OneDrive](https://support.microsoft.com/office/block-downloads-for-view-only-files-in-sharepoint-and-onedrive-6051184b-62ac-4149-b874-13dcd40ef91e).

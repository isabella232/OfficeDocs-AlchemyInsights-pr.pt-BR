# <a name="find-a-specific-quarantined-message"></a>Encontrar uma mensagem em quarentena específica

1. Vá para o [Centro de Conformidade & segurança.](https://go.microsoft.com/fwlink/p/?linkid=2077143)
2. Vá para **Quarentena de Revisão de Gerenciamento**  >  **de**  >  **Ameaças.**
3. Escolha **Pesquisar**.
4. Escolha qualquer combinação de condições selecionando as caixas de seleção ao lado delas (não é possível usar caracteres curinga no momento).

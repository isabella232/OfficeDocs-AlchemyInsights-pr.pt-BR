# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Integração de um Windows para o Microsoft Defender para Ponto de Extremidade

Para o sistema operacional Windows Server, o Microsoft Defender for Endpoint fornece recursos avançados para detectar e investigar ataques. Esses recursos são fornecidos perfeitamente, por meio do Central de Segurança do Microsoft Defender console.

O Defender for Endpoint dá suporte às seguintes versões do Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Servidor (SAC) versão 1803 e posterior
- Windows Server 2019 e posterior
- Windows Edição principal do Server 2019

Para saber mais, consulte [Onboard Windows para o serviço microsoft Defender for Endpoint](https://go.microsoft.com/fwlink/?linkid=2143627).

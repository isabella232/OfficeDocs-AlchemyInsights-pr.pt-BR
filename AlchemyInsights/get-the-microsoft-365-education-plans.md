# <a name="get-the-microsoft-365-education-plans"></a>Obter os planos do Microsoft 365 Education

Se o seu instituto já estiver qualificado para o Microsoft 365 Education, você poderá adicionar a assinatura A1, A3 ou A5 a partir do **Centro de administração do Microsoft 365**. 

1. Vá para **Faturamento > [Produtos e serviços](https://go.microsoft.com/fwlink/p/?linkid=868433)**.

2. Use a caixa de Pesquisa para pesquisar em "A1", "A2" ou "A3".

3. Selecione o número de usuários que usarão o plano.

Se o seu instituto ainda não configurou o Microsoft 365 Education, mas você está tentando adicionar o Microsoft 365 gratuitamente, [comece](https://www.microsoft.com/education/products/office) fornecendo o seu email.

 Para comparar planos, confira os planos [Para seus alunos](https://www.microsoft.com/microsoft-365/academic/compare-office-365-education-plans?activetab=tab:primaryr1) e [Para professores, docentes e funcionários](https://www.microsoft.com/microsoft-365/academic/compare-office-365-education-plans?activetab=tab:primaryr2).

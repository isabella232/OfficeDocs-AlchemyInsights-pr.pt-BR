# <a name="ignorelegalhold-switch"></a>**–IgnoreLegalhold** alternar

O **–IgnoreLegalhold**, utilizado no seguinte commandlet, não está disponível ou funcional para os objetos do usuário no Exchange Online.

- Disable-mailbox
- Remove-mailbox
- Disable-mailuser
- Remove-mailuser

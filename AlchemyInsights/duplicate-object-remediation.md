# <a name="duplicate-object-synchronization-errors"></a>Erros de sincronização de objetos duplicados

Se você receber um dos seguintes erros durante uma sincronização:

- Um objeto sincronizado com o mesmo endereço proxy já existe no diretório de Microsoft Online Services.

- Uma mensagem semelhante para UserPrincipalName.

Use o novo diagnóstico de Saúde do AADConnect para resolver erros de objeto duplicados. Para obter mais informações, consulte [Diagnostic and remediate duplicated attribute sync errors](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

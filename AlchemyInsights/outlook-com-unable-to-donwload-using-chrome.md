# <a name="cant-download-files-in-chrome"></a>Não é possível baixar arquivos no Chrome

**Problema** Estamos cientes de um problema que impede que alguns clientes baixem arquivos ao usar o navegador do Google Chrome. 

**Solução alternativa** Use [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) ou outro navegador da Web.
Saiba mais sobre como [baixar arquivos e anexos Outlook.com.](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)

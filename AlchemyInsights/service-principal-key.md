# <a name="service-principal-key"></a>Credencial de entidade de serviço

Para a orientação sobre o gerenciamento de credenciais para uma entidade de serviço, confira os seguintes artigos:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) – Para obter as credenciais para uma entidade de serviço usando o PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) – Para criar uma nova credencial para uma entidade de serviço usando o PowerShell.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) – Para remover uma credencial de uma entidade de serviço usando o PowerShell.

# <a name="microsoft-stream-licensing-overview"></a>Visão geral do Licenciamento do Microsoft Stream

O Microsoft Stream é um serviço de nuvem licenciado por assinatura de usuário. Ele está disponível como parte das assinaturas qualificadas do Microsoft 365. Consulte Preços do Microsoft 365 para obter informações sobre cada tipo de assinatura.

- [Visão geral do Licenciamento do Microsoft Stream](https://docs.microsoft.com/stream/license-overview)

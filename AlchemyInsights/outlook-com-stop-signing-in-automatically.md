# <a name="how-to-stop-signing-in-automatically"></a>Como parar de entrar automaticamente

Para parar de entrar em Outlook.com automaticamente, desmarque a caixa para **Manter-me** ingressado na página onde você inscreva sua senha.
  
Para saber mais, confira Como entrar ou sair [de Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).

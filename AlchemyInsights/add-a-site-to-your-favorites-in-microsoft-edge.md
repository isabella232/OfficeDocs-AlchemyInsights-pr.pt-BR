# <a name="add-a-site-to-your-favorites-in-microsoft-edge"></a>Adicione um site aos seus favoritos no Microsoft Edge

1. Em Microsoft Edge, vá para o site que você deseja adicionar.
2. Na barra de endereços, selecione o **ícone Adicionar esta página aos favoritos.** Como alternativa, selecione **Ctrl+D**.
3. Opcional: na caixa **de diálogo Editar** favorito, renomeie o favorito e escolha uma pasta diferente para salvá-lo.
4. Em seguida, selecione **Feito** na caixa **de diálogo Editar** favorito.

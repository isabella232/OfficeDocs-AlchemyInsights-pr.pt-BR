# <a name="sharepoint-online-throttling"></a>Otimização do SharePoint Online

SharePoint Online usa a limitação para manter o melhor desempenho e confiabilidade do serviço SharePoint Online. A limitação limita o número de ações do usuário ou de chamadas simultâneas (por script ou código) a fim de impedir o uso excessivo de recursos. Se você for estrangulado, 99% das vezes é por causa do código personalizado.
  
Para obter mais informações sobre a aceleração, consulte Evite [ser barrado](https://go.microsoft.com/fwlink/?linkid=2022019) ou bloqueado no SharePoint Online

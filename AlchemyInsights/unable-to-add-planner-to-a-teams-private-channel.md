# <a name="unable-to-add-planner-to-a-teams-private-channel"></a>Não é possível adicionar o Planner a um canal privado do Teams

Atualmente, os canais privados do Microsoft Teams não têm suporte para adicionar o Planner.  Estamos trabalhando no suporte total de aplicativos para canais privados, incluindo extensões de mensagens e bots. Confira [Canais privados no Microsoft Teams](https://docs.microsoft.com/microsoftteams/private-channels#what-you-need-to-know-about-private-channels) para saber mais.
